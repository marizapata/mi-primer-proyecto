# Galería de Perros

Galería web de fotos de perros desarrollada con HTML y CSS, con diseño en grid y efectos visuales.

## Contenido

- Página principal (index.html)
- Estilos CSS para diseño y animaciones
- Galería de imágenes organizada en grid
- Tarjetas con efectos hover

## Cómo usar

Abre el archivo index.html en tu navegador para ver la galería.
